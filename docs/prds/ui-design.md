# Maestro — UI 与交互设计

> 定义 Maestro 插件的界面布局、交互模式和视觉风格。
> 最后更新：2026-03-12

---

## 整体布局

Maestro 占据 VS Code 的左侧边栏（Activity Bar + Sidebar），不占用编辑器区域。

```
┌──────────────────────────────────────────────────────┐
│  Activity Bar  │     Sidebar (Maestro)    │  Editor  │
│                │                          │          │
│  [📁 Explorer] │  ┌──────────────────┐   │          │
│  [🔍 Search  ] │  │  View Switcher   │   │          │
│  [🎵 Maestro]◀─│  │  [Docs][Tasks]   │   │          │
│  [⚙ Settings] │  │  [Context][Dash]  │   │          │
│                │  ├──────────────────┤   │          │
│                │  │                  │   │          │
│                │  │  Active View     │   │          │
│                │  │  Content Area    │   │          │
│                │  │                  │   │          │
│                │  │                  │   │          │
│                │  └──────────────────┘   │          │
└──────────────────────────────────────────────────────┘
```

## 视图结构

Maestro 在侧边栏中提供 4 个主视图，通过顶部 Tab 或 Activity Bar 子图标切换：

### 视图 1: Docs（文档导航）

实现方式：VS Code TreeView API（原生体验）

```
📋 Docs
├── 📂 me2ai（人维护的文档）
│   ├── 📂 PRDs
│   │   ├── 📄 overview.md
│   │   └── 📄 ui-design.md
│   ├── 📂 Tech
│   │   └── 📄 architecture.md
│   ├── 📄 AGENTS.md
│   └── 📂 Rules
│       ├── 📄 ai-boundary-framework.mdc
│       └── 📄 project-methodology.mdc
│
└── 📂 ai2ai（AI 维护的文档）
    ├── 📄 status.md          [🟢 就绪度标记]
    ├── 📄 decisions.md       [12 条记录]
    ├── 📂 Tasks
    │   └── 📄 001-skeleton.md [✅ 已关闭]
    └── 📂 Iterations
        └── 📄 001-skeleton.md
```

交互：
- 单击：在编辑器中打开文件
- 右键菜单：复制文件路径、复制文件内容（用于粘贴到 AI 对话）
- ai2ai 文件旁显示状态徽标（任务数、记录数等）

实现方式：Webview（React）

│  进度: ████████░░ 5/8       │
├─────────────────────────────┤
│                             │
│  ☐ T1: 插件脚手架搭建       │
│    验证: 能 F5 启动调试      │
│                             │
│  ☑ T2: TreeView 注册        │  ← 已完成，灰色
│    验证: 侧边栏可见          │
│                             │
│  ◉ T3: 文档解析逻辑         │  ← 进行中，高亮
│    验证: 能列出文档树        │
│                             │
│  ☐ T4: 点击跳转             │

│                             │

交互：
- 点击 checkbox：切换任务状态（自动更新 task 文件）
- 点击任务标题：在编辑器中打开对应 task 文件并跳转到该任务行
- 进度条：实时反映完成比例

## 视觉风格

- 遵循 VS Code 主题变量（`--vscode-*` CSS 变量），自动适配亮色/暗色主题
- TreeView 部分使用原生 VS Code 样式，零自定义
- Webview 部分使用 VS Code 的设计语言，不引入独立的设计系统
- 图标使用 VS Code 内置的 Codicon 图标库

## 命令面板（Command Palette）

注册以下命令供用户通过 `Ctrl+Shift+P` 调用：

| 命令 | 描述 |
|------|------|
| `Maestro: Initialize Project` | 在当前工作区创建 Maestro 文档目录结构 |
| `Maestro: Copy Context to Clipboard` | 生成上下文并复制到剪贴板 |
| `Maestro: Show Task Board` | 打开任务看板视图 |
| `Maestro: Refresh` | 刷新所有视图数据 |
